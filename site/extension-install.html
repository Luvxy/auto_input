크롬 확장프로그램 설치

현재 MVP 확장프로그램은 Chrome Web Store 등록 전이므로 개발자 모드로 설치합니다.

설치 순서

  1. GitHub 저장소에서 Auto Input 코드를 내려받습니다.
  2. 크롬 주소창에 chrome://extensions를 입력합니다.
  3. 오른쪽 위 개발자 모드를 켭니다.
  4. 압축해제된 확장 프로그램을 로드합니다를 누릅니다.
  5. 프로젝트 폴더 안의 extension 폴더를 선택합니다.
  6. Auto Input PC 앱을 실행한 뒤 웹페이지에서 확장프로그램의 매핑 모드를 켭니다.

연결 확인

PC 앱 상단에 Bridge ON이 표시되면 확장프로그램과 앱이 연결된 상태입니다.
