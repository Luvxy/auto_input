Features

처음 쓰는 사람도 따라갈 수 있는 자동화 흐름

크롬에서 직접 매핑

확장프로그램의 매핑 모드를 켜고 웹페이지의 입력창이나 버튼을 클릭하면 PC 앱에 자동 등록됩니다.

CSV 컬럼 연결

고객명, 전화번호, 주소 같은 CSV 컬럼을 웹 입력창과 연결해 행 단위로 반복 실행합니다.

테스트 실행

전체 실행 전에 첫 번째 행만 실행해 입력과 클릭 순서가 맞는지 확인할 수 있습니다.

실행 로그

각 행마다 어떤 값이 입력됐고 어떤 버튼이 눌렸는지 로그로 확인할 수 있습니다.

How it works

자동화는 네 단계로 끝납니다

  1. 웹 요소 매핑 크롬 확장프로그램으로 입력창과 버튼을 선택합니다.
  2. CSV 불러오기 반복 입력할 데이터를 PC 앱에서 확인합니다.
  3. 단계 구성 값 입력, 클릭, 대기 순서를 원하는 대로 배치합니다.
  4. 실행 테스트 실행 후 전체 데이터를 반복 처리합니다.
